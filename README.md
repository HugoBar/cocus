Cocus is a Rails web app that helps you manage your recipes and fridge inventory. Track ingredients, check which recipes you can cook, and adjust quantities after cooking.

---

## Key Features

- **Recipe Management:** Create, view, update, and delete recipes with detailed steps, prep times, and servings.  
- **Ingredient Inventory:** Manage your fridge inventory including quantities and categories.  
- **Recipe Availability Check:** Quickly see which recipes you can cook with available ingredients.  
- **Missing Ingredient Detection:** Automatically flags missing or insufficient ingredients for any recipe.  
- **Cooking Mode:** Deduct ingredients after cooking and allow manual adjustments.
